什么是GitHub

GitHub是一个基于Git的代码托管平台，它允许开发者存储、管理和分享代码。除了代码托管功能外，GitHub还提供了项目管理、问题跟踪、代码审查等功能，是全球最流行的开源项目托管平台之一。

注册GitHub账号

  1. 访问GitHub官网：https://github.com
  2. 点击右上角的"Sign up"按钮
  3. 填写用户名、邮箱和密码
  4. 完成验证步骤
  5. 验证邮箱后，账号注册完成

创建仓库

  1. 登录GitHub后，点击右上角的"+"图标，选择"New repository"
  2. 填写仓库名称（例如：my-first-repo）
  3. 添加仓库描述（可选）
  4. 选择仓库可见性（Public或Private）
  5. 勾选"Initialize this repository with a README"
  6. 点击"Create repository"按钮

基本操作

克隆仓库

将远程仓库复制到本地：

git clone https://github.com/你的用户名/你的仓库名.git

提交更改

  1. 在本地修改文件
  2. 添加更改到暂存区：git add .
  3. 提交更改：git commit -m "提交信息"
  4. 推送到远程仓库：git push

拉取更新

获取远程仓库的最新更改：

git pull

GitHub Pages

GitHub Pages允许你直接从GitHub仓库托管静态网页。要启用GitHub Pages：

  1. 进入仓库页面，点击"Settings"
  2. 在左侧导航栏中选择"Pages"
  3. 在"Source"部分，选择分支（通常是main或master）
  4. 选择根目录或/docs文件夹
  5. 点击"Save"，几分钟后你的网站将在https://你的用户名.github.io/你的仓库名/可用
